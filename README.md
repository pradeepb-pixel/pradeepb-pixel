## About

AI Engineer with 2 years of experience building and deploying production-grade LLM applications inside financial services and healthcare organizations. My work spans the full lifecycle  scoping projects with stakeholders, building Claude-powered agents and RAG pipelines, training teams to own the tools, and writing documentation that survives handoff.

I have shipped multi-agent systems at Visa, AI customer service tooling at Citi Bank, and forecasting infrastructure at CVS Health. I work well with both technical and non-technical audiences and care about building things that continue working after I leave.

Currently open to AI Engineer and Data Scientist roles.

## Experience

**Data Scientist / AI Engineer — Visa**
`May 2025 – Present`

Embedded with the financial crime and compliance team to scope and build AI solutions for fraud detection and regulatory workflow automation. Worked directly with business stakeholders to define sprint outcomes. Built a multi-agent system using LangGraph and LangChain on AWS Bedrock that coordinates fraud scoring, compliance validation, transaction monitoring, and audit reporting. Developed RAG architecture using AWS Bedrock Knowledge Bases and Claude API for contextual document retrieval across compliance policy repositories. Ran internal training sessions for engineering and compliance staff.

Built NLP classification models for transaction categorization and risk-scoring pipelines for credit operations. Worked with non-technical stakeholders to define success metrics and built dashboards in Power BI and Tableau to track model performance.

---

## Projects

**Multi-Agent Fraud Detection System**
LangGraph-based multi-agent platform on AWS Bedrock. Four specialized agents  fraud detector, compliance validator, transaction monitor, and reporting agent  coordinate through a shared state graph to analyze transactions in real time. Built and deployed at Visa.
[github.com/pradeepb-pixel/multi-agent-fraud-detection](https://github.com/pradeepb-pixel/multi-agent-fraud-detection)

---

**Enterprise Compliance RAG Platform**
Retrieval-Augmented Generation system using Claude API and AWS Bedrock Knowledge Bases. Enables compliance teams to query thousands of regulatory documents in natural language and receive cited, grounded answers. Significantly reduced manual document search time.
[github.com/pradeepb-pixel/enterprise-compliance-rag](https://github.com/pradeepb-pixel/enterprise-compliance-rag)

---

**AI Customer Service Assistant**
Banking chatbot built with AWS Lex and LLM integration for Citi Bank. Handles account queries, transaction disputes, and card services. Includes LLM fallback layer for complex queries and seamless handoff to human agents when needed.
[github.com/pradeepb-pixel/ai-customer-service-assistant](https://github.com/pradeepb-pixel/ai-customer-service-assistant)

---

**Demand Forecasting Platform**
End-to-end forecasting pipeline on GCP Vertex AI for CVS Health supply chain. Ingests historical sales data from BigQuery, engineers time series features with Spark, trains models on Vertex AI, and generates batch predictions across thousands of SKUs with automated low-stock alerting.
[github.com/pradeepb-pixel/demand-forecasting-platform](https://github.com/pradeepb-pixel/demand-forecasting-platform)

---

**AI Resume Analyzer**
Working Claude API application — analyzes resume text and returns structured insights including skills extraction, job match scoring, improvement areas, and role recommendations. Includes a web interface built with Flask.
[github.com/pradeepb-pixel/ai-resume-analyzer](https://github.com/pradeepb-pixel/ai-resume-analyzer)
